Frequently asked questions

How do you calculate AI tool ROI with hours and a loaded rate?

Multiply hours saved per year by fully loaded hourly rate for annual value from time, subtract annual software cost, then divide net value by that cost for ROI.

What if value is partly revenue and partly time?

Add revenue lift into a separate scenario or sum only if finance certifies both as incremental—double-counting is common when teams mix top-line and efficiency stories.

Does this cover GPU or API overages?

Include predictable overages in annual software cost. Spiky experimental usage should be modeled separately so production ROI stays stable.
